LCEMP Overview
LCEMP is a fork of the Minecraft Legacy Console Edition source that enables LAN multiplayer hosting.
Features
  • Fully working multiplayer (Breaking blocks, placing blocks)
  • Kick system (Up to 8 players, modifiable in source)
  • Keyboard and mouse support
  • Gamma fix & Fullscreen support
  • The source code is available on the GitHub repositories.
  • To play with others, it is recommended to use Hamachi or Radmin VPN.

Setup Guide

Requirements

  • Download Radmin VPN
  • Ensure your firewall is configured correctly so the game can communicate over the network.

Singleplayer

To play singleplayer, simply launch the LCEMP executable (Minecraft.Client.exe).

Multiplayer Setup

  1. All players must download and install Radmin VPN.
  2. Create or join a Radmin network with the players you want to play with.
  3. The host should open the world they want to play.
  4. Other players can join using the host's Radmin IPv4 address.

DLC Installation Steps

  1. Download the DLCs provided above.
  2. After downloading, open the .zip file. You should see several folders inside.
  3. Press Ctrl + A to select all folders, then press Ctrl + C to copy them.
  4. Open the root directory of your LCEMP installation (the folder where Minecraft.Client.exe is located).
  5. Navigate to: \Windows64Media\DLC.
  6. Inside the DLC folder, press Ctrl + A and delete the existing folders/files.
  7. Press Ctrl + V to paste the folders you copied from the .zip.
  8. Once the folders are pasted into the DLC directory, launch Minecraft.Client.exe.
  9. The DLCs should now be installed.

Joining a Multiplayer Server

Launch the game with the IP launch argument:

-ip 123.123.123.123

Replace the IP with the host's Radmin IPv4 address.

Setting Your Username

You can set your in-game username using the -name launch argument.

Example: -name Simon

If you do not specify a name, the game will use your computer's username automatically.

How to Use Launch Arguments (Windows)

  1. Right-click the Minecraft.Client.exe
  2. Click Create Shortcut
  3. Right-click the shortcut and open Properties
  4. Add the launch arguments at the end of the Target field

Example: Minecraft.Client.exe -name Simon -ip 123.123.123.123
